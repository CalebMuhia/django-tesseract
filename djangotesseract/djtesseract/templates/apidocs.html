POST Parameters

Parameter Name Required Default Description
file True N/A Image/PDF file to be uploaded

Sample Response

{
	'result': "This is the resulting text from the image you uploaded"
}

Description

Endpoint for OCR. Post an image in a POST request with Content-type: application/x-www-form-urlencoded and receive back a response with the text found in that image/PDF file, as provided by Tesseract-OCR.

A successful request will receive a 200 status code and have a "result" attribute in the returned JSON dictionary. A failed request will receive a 400 status code and have a "message" attribute in the returned JSON dictionary with the reason for the failure.
